# Đồ án môn Mẫu thiết kế.
Link Github : https://github.com/PhuongPham1203/Mau-thiet-ke-doan

Source Code đồ án được lưu tại : .\source\doan
Yêu cầu Unity3d phiên bản 2019.4.15f1 hoặc cao hơn để chạy
Target Platform là Windows 64-bit

Toàn bộ code về các mẫu thiết kế được lưu tại : .\doan\Assets\Scripts

Cách điều khiển nhân vật trong game:
A : trái
D : Phải
1 : Hồi máu
J : nhảy
K : Bắn
L : Đổi vũ khí
